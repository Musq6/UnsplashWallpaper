## Unsplashwallpaper🖼️

## 主要功能

- 随机设置壁纸
- 使用壁纸合集
- 定时自动切换
- 收藏

## 须知

- 使用UnsplashAPI请求信息，需要Accesskey填入程序。免费申请使用，但限制50次请求/小时 申请地址：https://unsplash.com/developers
- 登录并创建app,信息随意填写，key在app详情页下方
- 请求数据都是通过API
- 会创建一个json文件本地保存设置信息
